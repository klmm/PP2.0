Sélectionnez votre Top10 pour chaque course

Avant chaque course, vous devez composer un Top10 sur le résultat final de cette course. Le pronostic se fait grâce à un glisser-déposer entre la zone des athlètes et celle de votre Top10.

Un nombre d'étoiles est attribué à chaque athlète. Allant de 0 à 3 pour les favoris, il est calculé en fonction des capacités de l'athlète. Si vous choisissez des athlètes avec peu d'étoiles, vous gagnez un bonus « prise de risque » qui sera répercuté sur votre score total.



Le comptage des points

Pour chaque pronostic, un nombre de points est attribué en fonction de plusieurs critères : position des athlètes, nombre d'athlètes trouvés et prise de risque. Le détail du calcul de points est disponible dans le Règlement.

Un classement interne à chaque pronostic est établi.



Les classements

Le classement général est établi à partir du nombre de points obtenus sur chaque pronostic. Le vainqueur de ce classement général est déclaré vainqueur du jeu.

Plusieurs autres classements sont également établis. Le détail de ces classements est également disponible dans le Règlement.



Echangez votre avis

Pour chaque pronostic, un espace "Commentaires" est ouvert. Vous pouvez ainsi donner votre avis avant, pendant et après l'étape.
