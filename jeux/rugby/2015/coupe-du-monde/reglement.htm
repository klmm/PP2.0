Ouverture d'un pronostic

La phase de poules est ouverte aux paris dès l'ouverture du jeu, le mercredi 16 septembre 2015 à 20h. Pour la phase finale, le pronostic est ouvert dès que les deux équipes sont connues.

Vous pouvez parier sur un match jusqu'au coup d'envoi. Une fois le pronostic fait, vous pouvez le modifier jusqu'à la date limite définie. Votre pronostic n'est pas visible par les autres joueurs avant cette date.

Le pronostic

Pour chaque match, vous devez pronostiquer sur le score de la rencontre ainsi que sur le nombre d'essais inscrits par chaque équipe. Pour les matches à élimination directe, les prolongations sont prises en compte dans le score final. Le comptage des points (voir section 3 du règlement) essaie de prendre en compte un maximum d'informations : vainqueur du match, écart, points et essais inscrits par chaque équipe...

Un coefficient est affecté à chaque rencontre selon son importance :

Matches de poule x1
Quarts x2
Demies et petite finale x3
Finale x4

Ce coefficient est pris en compte dans les classements Général et Phase finale (voir ci-après).

Le comptage des points

Vainqueur

Si vous avez trouvé le bon vainqueur, 100 points vous sont accordés. Dans le cas d'un match nul, vous n'obtenez ces 100 points uniquement si vous aviez pronostiqué un match nul.

Ecart

Si vous trouvez l'écart exact entre les deux équipes (avec le bon vainqueur), vous marquez un bonus de 40 points. Pour chaque point « d'erreur », vous perdez trois points.

Exemple : vous pronostiquez sur la victoire de l'équipe A avec +15 points. L'équipe A gagne avec +8 points. Vous gagnez tout d'abord les 100 points car vous avez trouvé le bon vainqueur. Puis vous obtenez un bonus de 40 - 3x7 = 19 points pour l'écart (car vous avez une erreur de 7 points sur l'écart).

Vous pouvez marquer des points « écart » sans avoir le bon vainqueur.

Exemple : vous pronostiquez sur la victoire de l'équipe A avec +6 points. L'équipe A perd avec -3 points. Vous ne gagnez pas les 100 points car vous n'avez pas trouvé le bon vainqueur. Mais vous obtenez un bonus de 40 - 3x9 = 13 points pour l'écart (car vous avez une erreur de 9 points sur l'écart).

Points inscrits

Si vous trouvez le bon nombre de points inscrits par une équipe, vous marquez un bonus de 15 points. Pour chaque point « d'erreur », vous perdez un point.

Exemple : vous pronostiquez que l'équipe A marque 28 points. L'équipe A marque 17 points. Qu'importe le vainqueur, vous obtenez un bonus de 15 - 11 = 4 points pour les points inscrits par A (car vous avez une erreur de 11 points sur ses points inscrits).

Essais inscrits

Si vous trouvez le bon nombre d'essais inscrits par une équipe, vous marquez un bonus de 15 points.

Exemple : vous pronostiquez que l'équipe A marque 3 essais. L'équipe A marque 3 essais. Qu'importe le vainqueur, vous obtenez un bonus de 15 points.

Les classements

Classement général

Le classement général est le classement principal du jeu, il reprend la somme des points obtenus lors des 48 matches de la compétition.

Classement Réussite

Ce classement est établi selon le nombre de fois où un joueur a pronostiqué le bon vainqueur.

Classement Phase finale

Ce classement prend uniquement en compte les matches de phase finale, à partir des quarts de finale.

Classement par points

Ce classement est établi selon la position obtenue pour chaque pronostic. Une victoire rapporte 25 points, une deuxième place 20 points, puis 16, 12, 10, 7, 5, 3, 2 et 1 point pour la dixième place.

Classement Homme du match

Ce classement est établi selon le nombre de fois où un joueur est désigné Homme du match (premier au classements des pronostics sur un match).

Classement Marqueur d'essais

Ce classement est établi selon le nombre de fois où un joueur a pronostiqué le bon nombre d'essais pour une équipe.
